Automatic Canteen Checkout

Computer Vision · Few-Shot Learning

The Problem

To increase checkout speed in a company canteen, I have developed a core computer vision algotithm for automatic food item recognition. The main challenge was the daily-changing menu, requiring a flexible few-shot detection approach that could adapt to new food items with minimal examples.

Approach

Using triplet loss together with hard-negative mining, I trained a convolutional neural network to produce embeddings for food item images. Every day, one image for each item in the menu was captured by the canteen staff. These images were used to create a reference database of embeddings. When a customer places their tray at checkout, the system captures an image of the tray, extracts embeddings for each visible food item, and compares them to the stored embeddings using nearest-neighbor search to identify each item.

What I Learned

Same as in Brain Atlas from MRI Scans, energy-based models got your back. The management was surprised that the computer vision algorithm can be that flexible.
